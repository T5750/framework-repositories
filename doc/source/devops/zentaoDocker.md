# Zentao Docker

默认latest为禅道开源最新版本

禅道版本和镜像tag对应关系如下：

禅道版本 | 镜像标签
---|---
开源版 | 以数字开头，如：12.3.3
专业版 | 以pro开头，如：pro8.8.3
企业版 | 以biz开头，如：biz3.7.2
