## Docker
```sh
docker run -d \
  --name=blender \
  -p 3000:3000 \
  -v /path/to/config:/config \
  linuxserver/blender
docker run -d --name=blender -p 3000:3000 linuxserver/blender
```
[http://localhost:3000/](http://localhost:3000/)
