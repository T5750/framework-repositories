# Ubuntu Docker

Ubuntu is a Debian-based Linux operating system based on free software.

## Docker
```sh
docker pull ubuntu:18.04
docker run -itd --name ubuntu18 ubuntu:18.04
docker run -itd --name ubuntu18 ubuntu:20.04
```
