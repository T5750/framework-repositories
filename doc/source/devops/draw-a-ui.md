# draw-a-ui

Turn Sketches into HTML Instantly

Use AI to turn your UI sketches into HTML code instantly, right from your browser.

## Getting Started
```sh
echo "OPENAI_API_KEY=sk-your-key" > .env.local
npm install
npm run dev
```
[http://localhost:3000/](http://localhost:3000/)
