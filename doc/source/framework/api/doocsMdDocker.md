# Doocs Md Docker

一款高度简洁的微信 Markdown 编辑器

## Demo
[在线编辑器地址](https://md.doocs.org/)

## Docker
```sh
docker run -d -p 8080:80 doocs/md:latest
```
[http://localhost:8080/](http://localhost:8080/)

## Runtime Environment
- [Vue.js](https://github.com/vuejs/vue)
- [TypeScript](https://www.typescriptlang.org/)
