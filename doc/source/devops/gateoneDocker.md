# Gate One Docker

Web Terminal Emulator and SSH Client

## Docker
```sh
docker run -d --name=gateone -p 443:8000 liftoff/gateone gateone
```
[https://localhost:443/](https://localhost:443/)

## Runtime Environment
- [Python 2.7.x](https://www.python.org/downloads/)
