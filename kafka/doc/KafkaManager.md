# Kafka Manager

## Configuration
`conf/application.properties`
```
kafka-manager.zkhosts="localhost:2181"
```

## Starting the service
```
bin/kafka-manager
```

### Windows
```
bin\kafka-manager.bat
```

[http://localhost:9000/](http://localhost:9000/)
