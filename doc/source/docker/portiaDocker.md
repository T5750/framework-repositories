# Portia Docker

Visual scraping for Scrapy

## Docker
```
docker run -d --name portia -v ~/portia_projects:/app/data/projects:rw -p 9001:9001 scrapinghub/portia
```
[http://localhost:9001/](http://localhost:9001/)
