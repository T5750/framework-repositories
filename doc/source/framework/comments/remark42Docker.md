# Remark42 Docker

Privacy-focused lightweight commenting engine

## Demo
[Demo site](https://remark42.com/demo/) available with all authentication methods, including email auth and anonymous access.

## Docker Compose
- `remark42.yml`
- [http://localhost:8080/web](http://localhost:8080/web)

## Setup on Your Website
```html
<script>
  var remark_config = {
    host: 'REMARK_URL',
    site_id: 'YOUR_SITE_ID',
  }
</script>
```
```html
<div id="remark42"></div>
```
