# RethinkDB Docker

The open-source database for the realtime web

## Docker
```sh
docker run -d -P --name rethinkdb rethinkdb
docker run -d -p 8080:8080 -P --name rethinkdb rethinkdb
```
[http://localhost:8080/](http://localhost:8080/)
