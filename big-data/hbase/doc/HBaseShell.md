# HBase Shell

## General Commands
- `status`
- `version`
- `table_help`
- `whoami`

## Data Definition Language
- `create`
- `list` - Lists all the tables in HBase.
- `disable`
- `is_disabled`
- `enable`
- `is_enabled`
- `describe`
- `alter`
- `exists`
- `drop`
- `drop_all`
- Java Admin API

## Data Manipulation Language
- `put`
- `get`
- `delete`
- `deleteall`
- `scan` - Scans and returns the table data.
- `count`
- `truncate`
- Java client API

## Starting HBase Shell
`./bin/hbase shell`
