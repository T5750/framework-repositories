# Chatbot Ollama Docker

Chatbot Ollama is an open source chat UI for Ollama.

## Docker
```sh
docker run -d --name chatbot-ollama -p 3000:3000 ghcr.io/ivanfioravanti/chatbot-ollama:main
```
[http://localhost:3000/](http://localhost:3000/)
