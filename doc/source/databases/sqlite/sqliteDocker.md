# SQLite Docker

SQLite is a C-language library that implements a small, fast, self-contained, high-reliability, full-featured, SQL database engine.

## Docker
```
docker run -it nouchka/sqlite3
docker run -it --rm --name sqlite nouchka/sqlite3
```
